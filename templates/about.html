DETECTING RANSOMWARE THREATS IN DISK STORAGE

A novel approach to countering ransomware involves bypassing conventional antivirus that ransomware often eludes. By encrypting files and effectively restricting users' access to their systems and data, ransomware poses a significant threat. A newly proposed ransomware detection system operates in virtual machines, gathering data on processor and disk I/O activities from the host machine.

Storage Attribute Information
Instructions Total number of fundamental operations executed by processor
LLC - Stores "Last Level Cache" refers To store number of right operations that occurs in LLC
L1-icache-load-misses Number of misses that occur in the Level 1 instruction cache (L1 iCache) when attempting to load instructions.
branch-load-misses Number of times the CPU attempted to load data from memory due to branch instructions and encountered a cache miss.
node-load-misses Performance metric related to cache misses in a computer system
rd_req "Read Request" refers to a request made by a component in a computer system to read data from a storage device, such as RAM or a hard drive
rd_bytes "Read Bytes" refers to metric used in performance monitoring or system analysis to measure the amount of data read from storage by a particular process
wr_req "Write Request" refers to a write request is a command or signal sent to storage devices to write data to a specific location in the storage medium
wr_bytes "write bytes" refer to the amount of data that has been written to storage, such as a disk or solid-state drive (SSD), during a specific operation
flush_operations "Flush Operatios" refer to the act of writing buffered data from a cache
rd_total_times Total number of read operations performed in a system
wr_total_times Total number of times data has been written to a storage device
flush_total_times Total number of times that data has been flushed from a cache or written to permanent storage.
